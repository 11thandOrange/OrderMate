📋 #62: Templates Pull from Settings

Templates load from Firebase via FirebaseConfigManager.getTemplates(merchantId).
Stored at: merchants/{merchantId}/templates

✨ #63 & #64: Template Auto-Fill

When a template is selected:
• Content auto-fills message box
• Subject auto-fills subject field (email mode only)
• Customer phone/email comes from order data, not template

📜 #65: Notification History

Each sent notification includes reference: "order-{orderId}" for tracking.
Notifications appear in Order History card via Bird API.
