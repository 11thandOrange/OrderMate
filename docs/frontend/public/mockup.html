Orders

The Orders API allows you to create, read, update, and delete orders for a merchant. Orders contain line items, customer information, and payment details.

On this page

GET List Orders GET Get Order POST Create Order PUT Update Order DELETE Delete Order

The Order Object

Attribute Type Description
id string Unique identifier for the order
currency string Three-letter ISO currency code
total integer Total amount in cents
paymentState string OPEN, PAID, PARTIALLY_PAID, etc.
title string Order title/name
GET /v3/merchants/{mId}/orders

List Orders

Retrieves a list of orders for the specified merchant.

Parameters

mId string required

Unique identifier of the merchant

limit integer

Maximum results to return (default: 100)

Response

Example Response 
{
  "elements": [
    {
      "id": "ABC123",
      "total": 2499,
      "paymentState": "PAID"
    }
  ]
}

Try it out

GET /v3/merchants/{mId}/orders

Code Examples

curl "https://api.clover.com/v3/merchants/{mId}/orders" \
  -H "Authorization: Bearer TOKEN"
