Frequently Asked Questions

Learn how to use OrderMate's features to streamline your order management and keep your team and customers in sync.

Item Level Notes

How do I add notes to individual items?

To add item-level notes:

  • Open an order from the order list or calendar view
  • Tap on any line item in the order
  • The item popup will appear with customizable widget fields
  • Fill in single-select, multi-select, or text fields as needed
  • Tap Save to apply the notes to that item
Where do item-level notes appear?

Item-level notes appear in several places:

  • Order Detail Page: Notes display as colored pill badges beneath each line item
  • Order List: Key notes show as pill badges on order cards for quick reference
  • Printed Receipts: Item notes print directly under each item when receipt printing is enabled
  • Register Overlay: When accessed from Clover Register, item notes are visible in the order summary
What types of widgets can I use for item notes?

OrderMate supports several widget types for item-level notes:

  • Single-Select: Choose one option from a list (e.g., Size: Small, Medium, Large)
  • Multi-Select: Choose multiple options (e.g., Toppings: Nuts, Sprinkles, Chocolate)
  • Text Box: Free-form text for special instructions or allergies
  • Calendar/Date Picker: Schedule dates for item-specific deadlines

Configure these widgets in Settings → Item Widgets.

Order Level Notes

How do I add notes to an entire order?

To add order-level notes:

  • Open an order from the order list or calendar
  • Look for the Order Notes section in the order detail page
  • Tap on any widget field to add information
  • Use the calendar picker to schedule pickup or delivery dates
  • Add text notes for delivery addresses, gift messages, or special instructions
Where do order-level notes appear?

Order-level notes are displayed in:

  • Order Cards: Key notes show as pill badges in the order list for quick scanning
  • Order Detail Header: Full notes display at the top of the order detail page
  • Calendar View: Orders with scheduled dates appear on the visual calendar
  • Printed Receipts: Order notes print in the header or footer of receipts
  • Customer Notifications: Order notes can be included in SMS/email templates
Can I schedule orders for pickup or delivery?

Yes! Use the calendar widget in order-level notes to schedule orders:

  • Add a Date/Time widget to your order-level configuration
  • Select the pickup or delivery date when editing an order
  • The order will automatically appear on the calendar view
  • Color-code pickup vs delivery orders in widget settings
  • Schedule automatic email reminders before the due date

Search & Filter

How do I search for orders?

Use the search bar at the top of the order list to find orders quickly:

  • Search by customer name or phone number
  • Search by order number (e.g., #1042)
  • Search by custom field values from your widgets
  • Results update in real-time as you type
What filters are available?

OrderMate provides powerful filtering options:

  • Payment Status: Open, Paid, Partially Paid, Partially Refunded, Refunded, Closed
  • Order Type: Filter by order types defined in Clover
  • Date Range: Today, This Week, This Month, or custom date range
  • Custom Fields: Filter by any single-select or multi-select widget values
How do I customize which filters appear?

Customize your filters in Settings:

  • Go to Settings → Filters
  • Toggle which filters appear in the filter bar
  • Enable "Show in Filters" for any widget to make it filterable
  • Reorder filters by drag-and-drop to prioritize the ones you use most
  • Hide filters you don't need to keep the interface clean

Settings

How do I create and manage notification templates?

Creating Templates:

  • Go to Settings → Templates
  • Tap + New Template
  • Enter a template name (e.g., "Order Ready", "Reminder")
  • Write your message using available data variables
  • Save the template for quick reuse

Available Data Variables:

  • {customer_name} - Customer's name
  • {order_number} - Order ID/number
  • {order_total} - Total amount
  • {pickup_date} - Scheduled pickup date
  • {store_name} - Your business name

Deleting Templates: Swipe left on any template and tap Delete, or tap Edit and select templates to remove.

What are default templates?

OrderMate comes with pre-configured default templates:

  • Order Ready: Notify customers their order is ready for pickup
  • Order Reminder: Send a reminder before the scheduled pickup/delivery
  • Thank You: Send a thank you message after order completion

Default templates can be edited but not deleted. Create custom templates for your specific needs.

How do I configure widgets?

Configure widgets for both item-level and order-level notes:

  • Go to Settings → Item Widgets or Settings → Order Widgets
  • Tap + Add Widget to create a new widget
  • Choose the widget type (Single-Select, Multi-Select, Text, Calendar)
  • Enter a label and options (for select widgets)
  • Choose colors for each option to make them visually distinct
  • Toggle Required if the field must be filled
  • Toggle Show in Filters to make widget values filterable
  • Drag widgets to reorder them
How do I set up scheduled reminders and receipt printing?

Scheduled Email Reminders:

  • Go to Settings → Notifications
  • Enable Scheduled Reminders
  • Set how many days/hours before the due date to send reminders
  • Choose the reminder template to use
  • Reminders are sent to the merchant email on file

Automatic Receipt Printing:

  • Go to Settings → Printing
  • Enable Auto-Print Before Due
  • Set how many minutes before the scheduled time to print
  • Select which printer to use
  • Receipts print automatically for upcoming orders

Register Integration

How does the Register overlay work?

The Register overlay provides quick access to OrderMate from Clover Register:

  • A floating button appears on Clover Register
  • Tap the button to open the OrderMate overlay
  • Add notes and schedule orders without leaving the checkout flow
  • Changes sync immediately with the main OrderMate app
  • Close the overlay to return to Register
How do I enable the Register overlay?

To enable the Register overlay:

  • Open OrderMate and go to Settings → Register Integration
  • Toggle Enable Register Overlay
  • Grant the required overlay permission when prompted
  • Open Clover Register - the OrderMate button will appear
  • Drag the button to reposition it on screen if needed
Does OrderMate sync with Clover in real-time?

Yes! OrderMate syncs directly with your Clover POS:

  • New orders from Clover Register appear instantly in OrderMate
  • Payment status updates (Paid, Refunded, etc.) sync automatically
  • Customer information is pulled from Clover's customer database
  • Order modifications are reflected across all devices
  • No manual refresh needed - data updates in real-time

Profile Settings

How do I customize my profile?

Personalize your OrderMate experience:

  • Go to Settings → Profile
  • Tap on your avatar to choose a new image or select from preset avatars
  • Choose an accent color that matches your brand
  • Your accent color is used for buttons, highlights, and UI elements
  • Profile settings sync across all devices using the same account
What accent colors are available?

OrderMate offers multiple accent color options:

  • Orange (default) - Warm and inviting
  • Purple - Professional and modern
  • Green - Fresh and natural
  • Blue - Calm and trustworthy
  • Red - Bold and energetic
  • Pink - Fun and friendly

The accent color applies to buttons, selection highlights, calendar events, and interactive elements throughout the app.

Do my settings sync across devices?

Yes, your configuration syncs across all devices:

  • Widget configurations - Item and order widgets sync automatically
  • Templates - Notification templates are available on all devices
  • Profile settings - Avatar and accent color preferences sync
  • Filter preferences - Your filter configuration is saved

Settings are stored securely and persist across app reinstalls and device changes when using the same Clover merchant account.
