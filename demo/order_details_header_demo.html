Order Details Header - Pill Wrapping Demo

Demonstrates how the "Order Details" header should handle multiple status pills.
The title should remain fixed width, and pills should wrap to the next row when space is limited.

❌ Current (Broken) BUG

Title shrinks when pills are added. Pills can't wrap.

LinearLayout (horizontal, no wrap)
Order Details OPEN UNPAID
Card content...

✅ Fixed (FlexboxLayout) SOLUTION

Title stays fixed. Pills wrap to next row when needed.

FlexboxLayout (flexWrap="wrap")
Order Details OPEN UNPAID
Card content...

📱 Android XML Implementation

<!-- Outer LinearLayout: keeps chevron fixed on right -->
<LinearLayout
    android:id="@+id/orderDetailsHeader"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:orientation="horizontal"
    android:gravity="center_vertical"
    android:paddingHorizontal="20dp"
    android:paddingVertical="16dp">

    <!-- Inner FlexboxLayout: icon + title + pills (wraps) -->
    <com.google.android.flexbox.FlexboxLayout
        android:layout_width="0dp"
        android:layout_height="wrap_content"
        android:layout_weight="1"
        app:flexWrap="wrap"
        app:alignItems="center"
        app:alignContent="flex_start">

        <!-- Icon - fixed, won't shrink -->
        <FrameLayout
            android:layout_width="32dp"
            android:layout_height="32dp"
            android:layout_marginEnd="10dp"
            app:layout_flexShrink="0"
            ... />

        <!-- Title - fixed width, won't shrink, pushes pills right -->
        <TextView
            android:layout_width="wrap_content"
            android:layout_height="wrap_content"
            android:layout_marginEnd="10dp"
            android:text="Order Details"
            app:layout_flexShrink="0"
            app:layout_flexGrow="1"
            ... />

        <!-- Status Pills - will wrap to next row if needed -->
        <TextView android:id="@+id/orderPlacedStatusValue"
            android:layout_marginEnd="6dp"
            android:layout_marginTop="4dp"
            android:layout_marginBottom="4dp"
            app:layout_flexShrink="0"
            ... />

        <TextView android:id="@+id/paymentStatusBadge"
            android:layout_marginEnd="6dp"
            android:layout_marginTop="4dp"
            android:layout_marginBottom="4dp"
            app:layout_flexShrink="0"
            ... />

        <TextView android:id="@+id/paymentTypeBadge"
            android:layout_marginEnd="6dp"
            android:layout_marginTop="4dp"
            android:layout_marginBottom="4dp"
            app:layout_flexShrink="0"
            ... />

    </com.google.android.flexbox.FlexboxLayout>

    <!-- Chevron - OUTSIDE FlexboxLayout, stays fixed on right -->
    <ImageView
        android:layout_width="20dp"
        android:layout_height="20dp"
        android:layout_marginStart="8dp"
        ... />

</LinearLayout>
