Parsing Logic

Input: "Pickup Date:Dec 25 • Status:Ready • Category:Birthday • Tags:Rush,VIP" Split by "•" → ["Pickup Date:Dec 25", "Status:Ready", "Category:Birthday", "Tags:Rush,VIP"] For each part, split by ":" → Label Value Widget Type Icon ───────────────────────────────────────────────────── Pickup Date → "Dec 25" → CALENDAR → 📅 Status → "Ready" → SINGLE_SELECT → ◉ Category → "Birthday" → MULTI_SELECT → 🏷️ Tags → "Rush,VIP" → MULTI_SELECT → 🏷️ (split by ",")
